1.Runing a Python Tkinter in Vscode

To run a tkiter code in a Vscode we need to select/convert python interpiture to python virtual interpiture to do that follow the following steps:
1.open the Vscode in floder
2.make one folder say f1
3.open the terminal of Vscode
4.In the terminal write: python -m venv f1
5.then activate popup click on "Yes"
6.then in terminal type .\f1\Scripts\activate

Example:


For creating vertiual environment 
            
                python -m venv f1

For activating environment 
            
                .\f1\Scripts\activate

2.Pack,grid in one progam

To solve this we can`t use "pack" and "grid" in one programe to use it we need to use pack to define frame and inidse that fram we can add "gride" comand and we need to use instade of tk window replace it with frame

Example

        
            import tkinter as tk
            root=tk.Tk()
            f=Frame(root)
            laber=Label(f,text="User Name")
            laber1=Label(f,text="Contect")
            laber.grid(row=0,colum=0)
            laber1.grid(row=1,colum=0)
            // to keep it in center
            f.pack()
            root.mainloop()
